# philosophers
Philosophers is a 42 School project focused on concurrency and synchronization in C. It solves the Dining Philosophers problem using threads, mutexes, and precise timing, aiming to prevent race conditions, deadlocks, and starvation while following the 42 Norm.

Soluction:
- Makefile and structure creation;
- Creation of the function that initialize the datas receive;
- Creation of the function that initialize the mutexes and philosofers;
- started the actions(eat, sleep, think), the routine(thread)  and the monitor(death verify);
